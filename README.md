# Extraxy

Smart Document Data Extraction

Extraxy is a premium SaaS platform for extracting structured data from documents using OCR, predefined templates, regex patterns, and rule-based parsing. It helps users organize invoices, receipts, bank statements, forms, contracts, resumes, and scanned PDFs without using AI-based processing.

## Features
- OCR text recognition
- Rule-based field extraction
- Template builder
- Regex-based parsing
- Bulk upload
- Search and filter
- CSV, Excel, JSON, XML, Google Sheets export
- REST API and webhook support
- Secure cloud storage
- Role-based access

## Tech Stack
- React
- Tailwind CSS
- Node.js
- Express.js
- Supabase
- Tesseract OCR
- Regex
- REST API
